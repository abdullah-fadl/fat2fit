# أسماء قواعد البيانات المحتملة لـ Odoo Cloud

بناءً على URL: `https://bperformance-scop.odoo.com`

## أسماء محتملة (جربها بالترتيب):

1. **`bperformance-scop-production`** ⭐ (الأكثر احتمالاً)
2. **`bperformance-scop-prod`** 
3. **`bperformance_scop_production`** (بشرطات سفلية)
4. **`bperformance_scop_prod`** (بشرطات سفلية)
5. **`bperformance-scop`** (بدون أي إضافات)
6. **`bperformance_scop`** (بشرطة سفلية)
7. **`bperformance`** (بدون -scop)
8. **`bperformance-scop-main`**
9. **`bperformance-scop-live`**

## كيفية معرفة الاسم الصحيح:

### الطريقة 1: من Odoo بعد تسجيل الدخول
1. سجل دخول إلى Odoo Cloud
2. اضغط على أيقونة المستخدم (في الأعلى يمين)
3. اضغط على "Preferences" أو "Preferences" 
4. ابحث عن "Database" - ستجد اسم قاعدة البيانات هناك

### الطريقة 2: من صفحة تسجيل الدخول
- إذا ظهرت قائمة اختيار قاعدة البيانات قبل تسجيل الدخول، هذا هو الاسم الصحيح

### الطريقة 3: من مدير Odoo
- اتصل بمدير Odoo Cloud واطلب منه اسم قاعدة البيانات

### الطريقة 4: التجربة
- جرب الأسماء المذكورة أعلاه واحداً تلو الآخر
- عندما يعمل الاتصال، يعني أنك وجدت الاسم الصحيح

## ملاحظة:
في Odoo Cloud، اسم قاعدة البيانات عادة يكون:
- نفس اسم الشركة/المشروع
- أو نفس اسم الشركة + `-production` أو `-prod`
- وأحياناً بشرطات سفلية `_` بدلاً من شرطات `-`
