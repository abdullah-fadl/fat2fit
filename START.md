# دليل البدء السريع

## 1. تشغيل النظام
```bash
npm run dev
```

## 2. إنشاء المستخدمين والباقات
افتح في المتصفح:
```
http://localhost:3000/api/setup
```

سيتم إنشاء:
- **مديرة:** admin@fat2fit.com / admin123
- **موظفة استقبال:** reception@fat2fit.com / reception123
- **الباقات الافتراضية**

## 3. تسجيل الدخول
افتح:
```
http://localhost:3000/login
```

استخدم:
- البريد: `admin@fat2fit.com`
- كلمة المرور: `admin123`

## 4. البدء في استخدام النظام
- تسجيل عميلات جديدة
- إنشاء اشتراكات
- تسجيل الدخول
- إصدار فواتير
- عرض التقارير
